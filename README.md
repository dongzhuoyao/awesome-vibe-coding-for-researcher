# Awesome Vibe Coding for Researchers

> Curated resources for AI-assisted research workflows, covering every stage of the research pipeline.

[![Awesome](https://awesome.re/badge.svg)](https://awesome.re)
[![Contributions Welcome](https://img.shields.io/badge/contributions-welcome-brightgreen.svg)](CONTRIBUTING.md)

"Vibe coding" is a development style where you collaborate with AI assistants to write code through natural language prompts and iterative refinement. This list organizes tools and workflows for applying vibe coding to academic and industry research.
